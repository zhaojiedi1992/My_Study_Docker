docker checkpoint 
===============================

docker checkpoint 是创建检查点的,实验功能。

创建用法
------------------------

.. code-block:: bash 

    docker checkpoint create [OPTIONS] CONTAINER CHECKPOINT

查看用法
------------------------

.. code-block:: bash 

    docker checkpoint ls [OPTIONS] CONTAINER

删除用法
------------------------

.. code-block:: bash 

    docker checkpoint rm [OPTIONS] CONTAINER CHECKPOINT
